# How it works

Builder's goal is to make it easy to design, build, and release software in the age of AI.
It has a focus on websites and web applications as the quickest way to live in an ever evolving
econosystem of new things.

Builder expects a couple of things:

- a `.builder.yaml` file with the configuration (see the
  [customization section](/customization/index/) for more info)
- a clean working tree
- willingness to create new tools

And that's it.
